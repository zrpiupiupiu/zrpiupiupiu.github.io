指针变量

  • 指针就是地址，而指针变量就是保存地址的变量
    • 普通变量的值是实际的值，指针变量的值是具有实际值的变量的地址

指针变量的声明

  • 对指针变量的声明和对普通变量的声明一样，唯一不同的是必须在指针变量名字前加上* 

    int *p;

​ 这个声明意思是p是指向int类型对象的指针变量

  • 指针变量可以和其他变量一起出现在声明中

    int i,*p,*q,a[20];

  • c语言要求每个指针变量只能指向一种特定类型的对象

    int *p;
double *q;
char *r;

  • 指针变量还可以指向另一个指针，成为指向指针的指针

取地址运算符

之前声明指针变量我们只是为指针留出空间，但还没有把它给对象

  • 初始化指针变量的方法

    • ```
      int i,*p;

      p = &i;

      
  通过把i的地址赋值给p，使得p指向i

+ ```
  int i;
  int *p = &i;

      在声明指针变量的同时对它进行初始化

间接寻址运算符

一旦指针变量指向了对象，我们就可以用*来访问对象中的内容，例如

int i=1;
int *p = &i;
printf("%d\n",*p);

我们将得到的不是i的地址，而是i的值1

  • 可以理解为*p就是i的别名

    • 如果对*p进行改变，随之i的值也会发生变化

      int i=1;
int *p = &i;
printf("%d\n",i);     /*1*/
printf("%d\n",*p);    /*1*/
*p = 2;

printf("%d\n",i);     /*2*/
printf("%d\n",*p);    /*2*/

字符串函数strcmp

int strcmp(const char *s1,const char *s2);
  • 用来比较两个字符串，返回：
    • 等于0 s1==s2
    • 大于0 s1>s2
    • 小于0 s1<s2

#include<stdio.h>
#include<string.h>

int mycmp(const char *s1,const char *s2)
{
	//int idx = 0;
	//whlie(s1[idx] == s2[idx]&&s1[idx]!='\0'){
	//idx ++;
	//}
	
	while(*s1 == *s2 && *s1!='\0'){
	s1++;
	s2++;
	}
	return *s1-*s2;
}

int main()
{
	char s1[] = "abc";
	char s2[] = "abc";
	printf("%d\n",mycmp(s1,s2));
	
	return 0;
}

比较两个字符串的大小，不能直接运用关系运算符，必须使用strcmp()函数

字符串函数strcpy

char *strcpy(char *restrict dst,const char *restrict src);

做的事情是：把第二个参数里头所表达的字符串拷贝到第一个参数所表达的那个空间里去

即把src的字符串拷贝到dst中

  • restrict表明src和dst不重合

  • 返回dst

字符串函数strcat()

char *strcat(char *restrict s1,const char *restrict s2)
  • 把s2拷贝到s1后面，接成一个长的字符串
  • 返回s1
  • s1必须有足够的空间
